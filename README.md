## 它到底是个啥

简单说，Paletro 给你 Mac 上几乎所有应用都装上了一个"命令面板"——就像 VS Code 里按 `⇧⌘P` 弹出来的那个东西，只不过它不限于某一个编辑器，而是覆盖整个系统。

它的原理挺巧：利用 macOS 的辅助功能（Accessibility），读取当前聚焦应用主菜单里的所有命令，然后让你用键盘搜索、过滤、执行。手都不用离开键盘，菜单栏那点事儿就全办了。我在 Finder 里整理文件、在 Typora 里写文档、在 Sketch 里做图，都靠它一键直达。说真的，用过之后你会觉得"用鼠标点菜单"这件事，突然变得很笨重。

## 几个让我离不开的功能

**键盘交互搜索**——按下快捷键唤出面板，输入关键字就能定位到具体菜单命令。模糊搜索很灵，打几个字母就能匹配到，不用记全名。

**拼音也能搜**——这个对我这种习惯中文输入的人太友好了。哪怕软件界面是英文的，我也能用拼音去定位汉化菜单，不用在脑子里先翻译一遍。

**灵活过滤规则**——可以针对特定应用禁用面板，或者排除不需要的菜单项。比如 Safari 把整个"历史"菜单都塞进命令列表里就没必要，排除掉之后搜索结果干净多了。

**自定义主题色**——预设好几款外观，还能跟着系统深色模式自动切换，看着不突兀。

**高级拓展**——你可以给应用加自定义执行命令，再绑个快捷键。等于在原有菜单之外，又给自己开了条"私人快捷通道"。
